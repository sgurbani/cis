PA2: Correcting Distortion and CT to Real World Registration

The purpose of this assignment was to update the code package to calculate a distortion function for the EM tracker space. Then, we use the undistorted space to register CT images w.r.t. real world using fiducials. The full details of the assignment can be found in this PDF document.
